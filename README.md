🌈 Rainbow After the Rain
An interactive web experience — tap an umbrella in the rain, watch a rainbow slowly fill the sky, and send a message to someone you miss.
→ Try it here
✨ Experience
Tap the screen to enter the world of rain
Tap the umbrella — the rain begins to stop
A rainbow gradually appears over 20 seconds
A gentle silhouette of someone you miss emerges in the light
Write a message and release it into the sky
🎧 Best with headphones
The rain uses binaural-style stereo audio — wearing headphones gives you a 3D immersive feeling, like standing in the rain yourself.
💡 Why I made this
"Whenever I see a rainbow, I think of someone."
I wanted to create a small, quiet space where anyone could visit that feeling — anytime, anywhere, from a single browser tab.
　🌠Built with
p5.js — graphics & animation
Web Audio API — rain sound & bell tones (binaural stereo)
Pure HTML / CSS / JavaScript — single file, no install needed
　 Use it
Just open the link in any browser. No app download, no sign-up.

Works on iPhone, Android, and desktop.
